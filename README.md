Load IPLs - based on https://github.com/ESX-PUBLIC/fivem-ipl converted to NFive

## Installation
Install the plugin into your server from the [NFive Hub](https://hub.nfive.io/Addemod/IPL-Loader): `nfpm install Addemod/IPL-Loader`
